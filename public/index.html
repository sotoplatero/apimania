APIMANIA

Create a pdf from a web.

Convert a web page into a pdf file passing it as a parameter the url address. Ex: /api/pdf?url=https://dev.to/

Take a screenshot of a web.

Ex: /api/screenshot?url=https://dev.to/

Create image from code

Create an image of the formatted code. At the moment only supports: javascript, typescript, json, yaml, css, scss, less, vue, html.

Parameters:

  • code: (required) Code you want to convert in image.
  • lang: (optional) Language of the code to convert. Default: javascript
Ex javascript: /api/screencode?code=if(true){alert('hola mundo!!')}
Ex json: /api/screencode?code={%22allOn%22:%20%22Single%22,%20%22Line%22:%20%22example%22,%22noSpace%22:true,%22quote%22:%20{%27singleQuote%27:%20%27example%27,%22indented%22:%20true,},%22phoneNumbers%22:%20[{%22type%22:%20%22home%22,%22number%22:%20%22212%20555-1234%22},{%22type%22:%20%22office%22,%22trailing%22:%20%22commas%20by%20accident%22},],}

APIMANIA

Create a pdf from a web.

Convert a web page into a pdf file passing it as a parameter the url address. Ex: /api/pdf?url=https://dev.to/

Take a screenshot of a web.

Ex: /api/screenshot?url=https://dev.to/

url url of the web page required
size Screenshot format: full full page, window only window, element css selector
/api/screenshot?url=https://dev.to&size=full
/api/screenshot?url=https://dev.to&size=.crayons-story--featured		 optional Default window

Convert text to image.

Create image with text content Ex: /api/txt2img?text=Create%20image%20with%20content%20text&size=7rem&format=16/9&color=red&bg=orange&font=Yellowtail

text Content text to show in image required
format Sets the aspect ratio of the image. Allowed values: 16/9 and 4/3. optionalDefault: empty value. Will set the width and height according to the size of the text
color Color of the text. Any css value is allowed. optionalDefault: #000
bg Background color. Any css value is allowed. optionalDefault: #fff
size Font size. Any css value is allowed. optionalDefault: 3rem
font Font family. Any family from google font. optionalDefault: arial

Create image from code. (LAB)

Create an image of the formatted code. Parameters:

  • code: Encode Code you want to convert in image.
  • lang: Language of the code: javascript, css, scss, json.
Ex: javascript, json, css, scss

Convert html tables to json.

Convert all tables on a web page to json format. Ex: /api/table?url=https://en.wikipedia.org/wiki/List_of_ISO_3166_country_codes

Convert web page to json.

Convert all web page to json format. Ex: /api/html2json?url=https://en.wikipedia.org/wiki/List_of_ISO_3166_country_codes

APIMANIA

Create a pdf from a web.

Convert a web page into a pdf file passing it as a parameter the url address. Ex: /api/pdf?url=https://dev.to/

Take a screenshot of a web.

Ex: /api/screenshot?url=https://dev.to/

Create image from code. (LAB)

Create an image of the formatted code. At the moment supports:javascript, json, css, scss, vue, html.

Parameters:

  • code: Code you want to convert in image.
  • lang: Language of the code to convert. Default: javascript
